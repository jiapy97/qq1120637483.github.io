心得体会：给图片加居中是没有效果的，可以给它的父级标签加居中。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style>
       body {
           /* 28px指的是行高 16px指的是字体的大小 */
           font: 16px/28px "微软雅黑";
       }
       h1 {
           text-align: center;
           font-weight: normal;
       }
       .gray {
           color: #888888;
            text-align: center;
            font-size: 15px;
            
       }
       a {
           text-decoration: none;
       }
       .search {
           color: #888888;
       }
       p {
           text-indent: 2em;
       }
    </style>
</head>
