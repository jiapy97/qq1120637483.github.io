## 需求
我们要生成2的n次方个01组合，以2的5为例我们需要生成32个不同的01组合。
## 代码
```c
#include <stdio.h>

int box[6];

void dfs(int step) {
    if(step == 6) {
        for(int i = 1; i <= 5; i++) {
            printf("%d", box[i]);
        }
        printf("\n");
        return;
    }
    for(int i = 0; i <= 1; i++) {
        box[step] = i;
        dfs(step + 1);
    }
}

int main() {
    dfs(1);
    return 0;
}
```

>生成2的6次方个，只用把step改为7 i<=6即可
