## 1：display
* display: none
隐藏所在元素，不占位置。

* display: block
转换为块级元素，同时显示元素。

## 2：visibility
* visibility: visible
元素可视

* visibility: hidden
元素隐藏，继续占有位置

## 3：overflow
* overflow: hidden
溢出隐藏

* overflow：scroll
溢出的部分可以使用滚动条进行滚动

* overflow: auto 
溢出的时候，显示滚动条，不溢出的时候，不显示滚动条。

## 代码展示
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .tudou {
            position: relative;
            width: 444px;
            height: 320px;
            background-color: aqua;
            margin: 30px auto;

        }
        .tudou img {
            width: 100%;
            height: 100%;
        }
        .mask {
            visibility: hidden;
            position: absolute;
            width: 100%;
            height: 100%;
            background: rgba(0,0,0,0.3) url(images/arr.png) center no-repeat
        }
        .tudou:hover .mask {
            visibility: visible;    
        }
    </style>
</head>
<body>
    <div class="tudou">
        <div class="mask"></div>
        <img src="images/tudou.jpg" alt="">
    </div>
</body>
</html>
```
