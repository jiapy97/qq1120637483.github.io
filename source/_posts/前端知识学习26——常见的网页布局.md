## 布局方式1
1：先采用标准流对大盒子进行布局。
2：大盒子内部的小盒子采用浮动进行布局。
## 案例
![](https://jiapeiyang.oss-cn-beijing.aliyuncs.com/img/20200617085954.png)

## 代码
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        * {
            margin: 0;
            padding: 0;
        }
        .top {
            height: 50px;
            background-color: aqua;
        }
        .banner {
            width: 1000px;
            height: 200px;
            background-color: blue;
            margin: 10px auto;
        }
        .box {
            width: 1000px;
            height: 200px;
            background-color: skyblue;
            margin: 10px auto;
        }
        .box li {
            list-style: none;
            width: 240px;
            height: 200px;
            background-color: red;
            float: left;
            margin-right: 13.3px;
        }
        .box .last {
            margin-right: 0;
        }
        .footer {
            height: 500px;
            background-color: violet;
            margin: 10px auto;
        }
    </style>
</head>
<body>
    <div class="top">top</div>
    <div class="banner">banner</div>
    <div class="box">
        <ul>
            <li>1</li>
            <li>2</li>
            <li>3</li>
            <li class="last">4</li>
        </ul>
    </div>
    <div class="footer">fooer</div>
</body>
</html>
```
