ES6——01.创建类和实例

constructor(构造函数)​

constructor()方法是类的构造函数（默认方法），用于传递参数，返回实例对象，通过new命令生成对象实例时，自动调用该方法。如果没有显示定义，类内部会总动给我们创建一个constructor().

  • 只要new对象，就会自动调用类的构造函数。
  • 类名首字母一般大写。

代码​

<script>
    class Star {
        constructor(name) {
            this.name = name;
        }
    }
    var test = new Star("测试名");
    console.log(test.name);
</script>
